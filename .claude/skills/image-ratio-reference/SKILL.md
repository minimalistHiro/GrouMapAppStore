# Image Ratio Reference

## 目的

画像や画像比率の依頼が来たときに、既存の比率ルールを参照し、未定義の比率は確認して追記する。

## 手順

1. 依頼が画像・画像比率に関係しているか確認する（画像サイズ、縦横比、アスペクト比などを含む）。
2. `IMAGE_RATIOS.md` を読み込む。
3. 依頼内容に該当する画像種別が **記載済み** の場合:
   - 記載されている比率を回答に反映する。
4. 依頼内容に該当する画像種別が **未記載** の場合:
   - どの比率にするかユーザーへ確認する。
5. 新しい比率が決まったら、`IMAGE_RATIOS.md` に追記する（既存の書式に合わせる）。

## 参照ファイル

- `IMAGE_RATIOS.md`

## 追記時のルール

- 既存フォーマット（箇条書き・全角コロン）に合わせる。
- 既存項目の意味を変えない。
- 不明点がある場合は追記せず、必ず確認する。
